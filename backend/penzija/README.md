# Backend (Sanity Studio)

Sanity Studio backend je u monorepo strukturi.

- Lokalni put: `backend/penzija`
- Root dokumentacija: `../../README.md`

## Komande

Iz ovog foldera:

```bash
npm run dev
npm run build
npm run start
npm run deploy
```

Ili iz root foldera projekta:

```bash
npm run dev:backend
npm run build:backend
npm run start:backend
```
