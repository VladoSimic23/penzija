# Frontend (Next.js)

Frontend aplikacija je u monorepo strukturi.

- Lokalni put: `frontend/my-app`
- Root dokumentacija: `../../README.md`

## Komande

Iz ovog foldera:

```bash
npm run dev
npm run build
npm run start
npm run lint
```

Ili iz root foldera projekta:

```bash
npm run dev:frontend
npm run build:frontend
npm run start:frontend
npm run lint:frontend
```
